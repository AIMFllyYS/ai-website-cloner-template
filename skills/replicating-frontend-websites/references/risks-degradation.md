# 通用风险表 · 降级矩阵 · 法律合规

> 合并自两次实战（静态站 R1–R10、WebGL SPA R1–R12）去重泛化。原则：**任何单点失败都不允许中断流程**——处置、记账、继续，最后统一汇报。

---

## §1 通用风险表

| # | 风险 | 信号 | 处置 |
|---|---|---|---|
| G1 | soft-404 污染镜像 | 缺失路径返回 200+HTML 外壳 | 指纹期标定签名；一切经 `fetch_ok`/`real_asset` 双判；假文件删除并记 `MISSING.log` |
| G2 | 发现方法选错（对 SPA 用 wget） | 镜像只有 1 个小 HTML | 回 Phase 0 重新指纹；按手册换 HAR 捕获 |
| G3 | 交互分支资源漏抓 | 命名空间计数为 0；离线时某状态缺料 | 回捕获阶段补对应交互（音频解锁/滚动/hover/路由） |
| G4 | 动态命名资源（序列帧等）边界未知 | JS 拼接文件名 | 权威参数优先；`probe_seq` 连续 miss 定界 + 安全阀 |
| G5 | 限速 / 封 IP / 盾挑战 | 429、403、challenge 页 | 真 UA、并发≤3、指数退避、断点续传；盾顽固时 headful 浏览器 |
| G6 | 中断/超时（大站长任务） | 下载中断 | `DONE.list` 续传设计，重跑即续 |
| G7 | wasm/worker/字体等关键依赖缺失 | 离线白屏/乱码/纹理黑块 | 手册 Phase 4 专项核验清单 |
| G8 | MIME/跨源头配置错误 | wasm 编译失败、SAB 报错 | 用 `scripts/serve.js`，不用默认静态服务器 |
| G9 | hash 文件名随重部署漂移 | 抓一半 404 突增 | 以当次抓到的入口为准；捕获后尽快下完；记录时间戳 |
| G10 | 特殊字符/中文路径 | 文件名乱码、404 | URL 编码下载、UTF-8 原名落盘、`--restrict-file-names=nocontrol` |
| G11 | 硬编码绝对域名逃逸 | 断网测试仍有线上请求 | grep 绝对 URL 清单；能改则相对化，不能改记已知差异 |
| G12 | 个别资源永久 404 | 重试后仍失败 | 同尺寸占位顶替 + `MISSING.log`，不停车 |
| G13 | 路由枚举不全（SPA） | 某页面从未被抓到 | sitemap + BFS 爬链 + pushState hook 三法并用 |
| G14 | 版权/品牌/第三方 IP | 站含字体、音乐、3D 资产授权 | 默认仅本地；公网部署前停下要授权（§3） |
| G15 | **过度宣称原始源码**（红线） | 把 Stage 2 的 T2-AI 产物说成"还原了原站源码" | 绝对禁止；交付必带 `playbook-reconstruction.md` §5 诚实声明，标注"等价重建≠原始源码" |
| G16 | cloner 环境不满足硬上 | Node<24 / 无浏览器访问 | 不启动 Stage 2，按 `playbook-reconstruction.md` §7 降级，交付 Stage-1 镜像并说明 |
| G17 | AI 重建非像素级 / 漏建动态逻辑 | 重建区块与线上有视觉差或交互缺失 | 逐区 QA diff mirror；偏差大复跑/修；不收敛→降级回交镜像，记已知差异 |
| G18 | live 改版/下线后仍当基准 | Stage 2 抓到的线上与 mirror 不一致 | 退化 mirror-only：以镜像快照为唯一基准重建+QA，交付标注基准为快照 |
| G19 | 等价重写"洗版权"误解 | 以为 AI 重建后即可自由商用 | 重写不消除目标站内容/设计版权；再发布仍需授权（§3） |

## §2 降级矩阵（从优到劣，逐级退）

| 受阻项 | 降级路径 |
|---|---|
| 无 wget | curl 循环（lib.sh 全覆盖，无损失） |
| 无 headless 浏览器环境 | 静态壳+已知资源直拉 + probe 探测（声明清单不保证全） |
| 序列帧/缩略图个别缺失 | probe 重定界 → 邻帧复制/灰占位 + 记账 |
| 视频/音频拉不到 | 空占位 + 海报帧 / 静音运行（视觉保全）+ 记账 |
| wasm 解码器缺失且补抓失败 | 纹理/几何降级为占位色，声明 L1 部分通过 |
| COOP/COEP 无法满足 | 确认是否真用 SAB；未用则去 COEP，用了则必须换服务器方案 |
| 某 JS 分支坏死 | 该区块降级为静态展示（内容保留、交互损失）+ 记账 |
| API 录制不全（hybrid） | 未录路径返回空态，禁止逃逸线上 |
| 全自动彻底失败 | 交付"已完成清单 + 缺口报告"，转人工分诊——这也是合法产出 |

## §3 法律合规（每次任务开场必念）

```
本次复刻默认用途：个人学习 / 技术研究 / 离线备份。
目标站点的品牌、设计、字体、图片、3D 资产、音乐可能受版权/商标保护。
再部署到公网、商用、或声称原创，均需取得权利方授权。
抓取遵守 robots.txt 礼仪、控制频率，不对源站造成压力。
```

- 部署前（Phase 7）**必须停下**向用户复述以上边界并取得明确确认。
- 用户要求公网部署而无授权依据时：照常完成本地镜像，部署步骤明确拒绝并说明原因。
